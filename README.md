# 🔁 C++ Loops — Practice Repository

A collection of small, independent C++ programs written while practicing **loops** (`for`, `while`) and iterative logic. Each file focuses on a single problem, covering number patterns, sums, counters, and star-pattern printing.

## 📚 About

This is not a single project — it's a growing set of standalone `.cpp` exercises, each demonstrating a specific loop-based concept: `for` loops, `while` loops, nested loops, increment/decrement logic, and iteration over ranges.

## 📁 What's Inside

**For Loop Basics**
- For Loop (general practice)
- For Loop — 10 Numbers Generation
- Print First 10 Natural Numbers
- Count 1 to 10

**While Loop Basics**
- While Loop (general practice)
- Decrement Using While Loop
- Table Using While Loop

**Number Ranges & Filters**
- Even Numbers Between 1 to 100
- Odd Numbers Between 1 to 50
- Total Even & Odd Numbers in 1 to 50
- Print Numbers Between 1 to 100, Skip 5

**Sum & Aggregation**
- Sum of Numbers 1 to 10
- Sum of Five Numbers (While Loop)
- Input Numbers and Print Sum
- Take 10 Numbers and Print Max & Min

**Sequences & Display**
- Display of Five Numbers (While Loop)
- Reverse Numbers
- Factorial Using Loop
- Multiplication Table (For Loop)
- Square (of numbers)

**Pattern Printing**
- Pattern For Loop
- Reverse Pattern For Loop
- Pakistan (flag/text pattern print)

Each `.cpp` file is self-contained and can be compiled and run independently. Pre-compiled `.exe` files are included for quick testing on Windows.

## 🎯 Purpose

- Practice writing and controlling `for` and `while` loops in C++
- Build comfort with iteration, counters, and loop conditions
- Strengthen logic building through small, focused problems
- Create a personal reference library of common loop patterns

## 🚀 Technologies Used

- C++
- Dev C++ / g++ compiler

## 🛠️ How to Run

```bash
g++ "filename.cpp" -o output
./output
```

Replace `filename.cpp` with the program you want to run. Each file is independent — no shared dependencies between them.

## 📌 Note

This repository is a practice log, not a finished application. It's continuously updated as new loop-based exercises are solved.
